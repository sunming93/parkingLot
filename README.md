###1.Parking Lot
| Given | When | Then|
|:-------|:------:|:-----:|
|有一辆车来了，有空余车位，车牌不重复|停车	|出票，停车成功|
|有一辆车来了，有空余车位，无车牌|停车|不出票，提示无车牌|
|有一辆车来了，有空余车位，车牌重复|停车|不出票，提示车牌重复|
|有一辆车来了，没有空余车位，|停车|不出票，提示没有空余车位|
|用户拿正确的票来取车（有相应的车）|取车|取车成功|
|用户拿错误的票来取车（没有相应的车）|取车|取车失败|
